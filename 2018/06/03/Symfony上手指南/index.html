Symfony是什么？

1. Symfony 是一套组件库

你的第一直觉可能觉得 Symfony 是一个 web 框架，但是从更底层的角度来说，Symfony 首先它是一套组件库，Symfony 组件库包含了高达30多个组件，这些组件不仅 Symfony 自己在用，很多 web 框架比如 Laravel 也在用。
